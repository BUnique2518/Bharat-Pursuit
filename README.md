# Bharat Pursuit Website

A modern, responsive website for Bharat Pursuit, a strategic and solutions marketing consulting firm specializing in global market positioning and go-to-market strategies.

## Features

- **Responsive Design**: Optimized for desktop, tablet, and mobile devices
- **Interactive Elements**: Hover effects and smooth animations
- **Fast Loading**: Built with Vite for optimal performance
- **SEO Friendly**: Proper meta tags and semantic HTML

## Technologies Used

- **React 18**: For building the user interface
- **Vite**: For fast development and building
- **CSS**: Custom styles with modern design principles
- **JavaScript (ES6+)**: For interactivity

## Getting Started

### Prerequisites

- Node.js (version 16 or higher)
- npm or yarn

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/BUnique2518/Bharat-Pursuit.git
   cd bharat-pursuit-website
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Start the development server:
   ```bash
   npm run dev
   ```

4. Open [http://localhost:5173](http://localhost:5173) in your browser.

### Build for Production

```bash
npm run build
```

The built files will be in the `dist/` directory.
